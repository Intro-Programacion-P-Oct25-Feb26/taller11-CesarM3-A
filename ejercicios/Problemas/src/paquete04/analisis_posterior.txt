Después de usar el programa, se observa que este guía al usuario con mensajes
claros en pantalla y le pide los datos necesarios para hacer el cálculo.
Según la opción elegida, solicita algunos valores adicionales importantes.
Con esos valores realiza una operación simple para obtener el monto a pagar.
Al final, muestra el resultado junto con el nombre y la cédula ingresados.
